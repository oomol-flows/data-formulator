# Data Formulator 数据公式化工具 📊

通过AI驱动的分析将您的数据转化为精美的可视化图表！Data Formulator是一个智能工作流平台，可帮助您分析电子表格数据并创建有意义的图表，无需任何编程知识。

## 什么是Data Formulator？

Data Formulator是一个无代码数据分析工具，可以自动处理您的Excel或CSV文件，并根据您的问题或目标生成交互式图表。只需上传您的数据，描述您想要分析的内容，让AI来完成繁重的工作！

## 核心功能

✨ **AI驱动分析** - 用纯中文理解您的数据分析目标  
📊 **自动图表生成** - 为您的数据创建最合适的可视化图表  
📁 **多文件支持** - 支持CSV和Excel文件  
🎨 **交互式可视化** - 生成可以交互和自定义的图表  
🔧 **无需编程** - 完美适合业务用户、研究人员和分析师

## 构建模块

Data Formulator由几个专门的模块组成，它们协同工作来分析您的数据：

### 📥 读取文件模块
**功能说明：** 将您的CSV或Excel文件加载到系统中  
**使用场景：** 总是从这里开始 - 这是将数据导入工作流的方式  
**特性：**
- 支持多种文件格式（CSV、XLSX）
- 自动数据预览
- 一次处理多个文件

### 🧠 AI数据分析模块（子流程）
**功能说明：** 系统的主要智能 - 根据您的指令分析数据  
**使用场景：** 加载文件后，这里就是魔法发生的地方  
**特性：**
- 理解自然语言指令
- 自动选择合适的分析方法
- 生成洞察并为可视化准备数据

### 🔍 数据衍生模块
**功能说明：** 从您的数据集中转换和提取特定洞察  
**使用场景：** 当您需要计算新指标或基于条件过滤数据时  
**特性：**
- AI驱动的数据转换
- 自定义计算和过滤
- 自动生成数据处理代码

### 📈 结果图表模块
**功能说明：** 从您分析的数据创建精美的交互式图表  
**使用场景：** 可视化结果的最后一步  
**支持的图表类型：**
- 柱状图
- 折线图
- 散点图
- 饼图
- 直方图
- 箱线图
- 面积图
- 雷达图
- 热力图
- 小提琴图
- 气泡图

### 📝 数据摘要模块
**功能说明：** 生成数据集的综合统计摘要  
**使用场景：** 当您需要数据关键统计信息的概览时  
**特性：**
- 自动统计分析
- 数据质量评估
- 字段架构生成

### 👁️ 代码预览模块
**功能说明：** 显示为您的分析自动生成的Python代码  
**使用场景：** 当您想了解或验证分析过程时  
**特性：**
- 清洁、可读的代码显示
- 学习数据分析的教育工具

## 如何使用Data Formulator

### 第1步：准备您的数据
- 将数据保存为CSV或Excel文件
- 确保数据有清晰的列标题
- 清理任何明显的数据质量问题

### 第2步：上传您的文件
1. 使用**读取文件模块**上传您的数据
2. 启用预览查看数据结构
3. 验证数据是否正确加载

### 第3步：定义您的分析目标
1. 将**AI数据分析模块**添加到您的工作流中
2. 将上传的数据连接到此模块
3. 用纯中文编写您的分析指令，例如：
   - "显示销售额与营销支出之间的关系"
   - "智商分数高于120的客户占多少百分比？"
   - "比较不同地区的表现"

### 第4步：生成可视化
系统将自动：
- 分析您的指令
- 相应地处理您的数据
- 生成最合适的图表类型
- 显示交互式结果

## 获得更好结果的技巧

### 编写好的指令
- **要具体：** 不要说"分析销售"，试试"显示2023年月度销售趋势"
- **包含背景：** 提及您要寻找什么 - 趋势、比较、分布
- **指定轴：** 如果您对X轴和Y轴有偏好，请提及它们

### 数据准备
- **清洁标题：** 使用清晰、描述性的列名
- **一致格式：** 确保日期、数字和类别格式一致
- **删除空行：** 上传前清理您的数据

### 图表选择
- **让AI选择：** 系统通常会自动选择最佳图表类型
- **需要时覆盖：** 如果您更喜欢特定的图表类型，可以指定
- **考虑您的受众：** 想想谁会查看这些图表
